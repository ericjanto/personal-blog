My personal website.

## Local Build

- Clone into local repo.
- Known to work with node version `14.4.0`
  - Use `nvm use 14` to point to that version
- *Make sure that `"gatsby-plugin-sharp": "^2.14.4"` in `package.json`
- Run `yarn install`
- Run `gatsby develop`
